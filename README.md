# KIBERS

Актуальные данные крипторынка.

##  Запуск

`make up`

или

`make run`

Разницу можно посмотреть в Makefile. После запуска будет доступен тут: http://localhost/

## Под капотом

* Laravel 8
* Vue 2.6
